# ATS Friendliness Score — Deferred

**Status: Backlog (not MVP)**
**Priority: Medium**

---

## Proposed

Add an ATS readability/friendliness score to CV and cover letter output.

## Proposed Metrics

- Keyword density (job keywords from posting found in CV)
- Format score (plain text vs tables/columns/headers — parsable format)
- Section presence (summary, experience, skills sections detected)
- Length score (word count within expected range)

## Why Deferred

- MVP goal: scoring/decision first, CV tailoring second
- ATS score is a quality-enhancement, not a blocker
- Needs user research to validate which metrics actually improve interview rates
- Not critical path for MVP launch

## When to Revisit

After MVP is live and Mic has real usage data — then validate which ATS factors actually matter for his target roles/companies.
